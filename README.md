# Evolutionary Path Planning for Small UAS in Wind Environments
MATLAB implementation of an Evolutionary Algorithm for pre-mission path 
planning of a fixed-wing sUAS through static obstacles in a steady wind field.
Developed as a final project for ASEN 5128 at the University of Colorado Boulder.

## File Structure
The FinalProject folder contains all new evolutionary path planning 
functions written for this project. The HomeworksAndHelpfulFunctions folder 
contains all functions made for the whole GNC framework including the Ttwistor aircraft 
parameters.
